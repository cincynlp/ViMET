# ViMET
Dataset for the EACL 2026 paper: A Computational Approach to Visual Metonymy

The full image repository is available for free at: _https://drive.google.com/drive/folders/1PzsympfVMW7UaQ08VNN0E-ws_PTVsc91?usp=share_link_

The image repository contains 1,000 stylistic and 1,000 naturalistic images.

The representamens for all the images are available in the Representamens folder and the mcq dataset is available in the MCQ_dataset folder, in .csv and .json format.

For any further questions, please contact ghosh2si@mail.uc.edu.
